# 🌤️ AP Weather Pro Dashboard
Dashboard cuaca modern dengan fitur pemetaan interaktif dan prakiraan 7 hari, dibangun dengan **JavaScript ES6** dan **Leaflet.js**.
https://github.com/user-attachments/assets/928d55c2-a0db-4cef-a961-bffaa6b1480a
## 🚀 Fitur Utama
* **Glassmorphism UI:** Desain modern menggunakan CSS Grid dan Flexbox.
* **Interactive Map:** Peta cuaca real-time menggunakan layer OpenWeatherMap.
* **Quick Access:** Akses cepat untuk kota-kota besar di Indonesia.
* **Detailed Metrics:** Informasi lengkap mulai dari suhu, kelembaban, hingga titik embun.

## 🛠️ Teknologi
* HTML5 & CSS3 (Custom Properties & Animations)
* JavaScript (Fetch API & Async/Await)
* Leaflet.js (Map Library)
* OpenWeatherMap API

## ⚙️ Cara Menjalankan Secara Lokal
1. Clone repository ini.
2. Dapatkan API Key gratis di [OpenWeatherMap](https://openweathermap.org/).
3. Buat file `config.js` dan masukkan key Anda:
   `const CONFIG = { API_KEY: 'YOUR_KEY_HERE' };`
4. Buka `index.html` di browser.
